## Пример использования

```
// ---запрос на сервер---
ServerRequest.execute(
    // адрес
    this.URL+'/store',
    // функция обработки ответа от сервера
    process,
        // тип запроса
    "post",
        // элемент для вывода информации
    this.msgElement,
    // данные запроса
    formData
);
```
