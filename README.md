## Пример использования

```
// ---запрос на сервер---
ServerRequest.execute(
    // адрес
    this.URL+'/store',
    // функция обработки ответа от сервера
    process,
    // тип запроса
    "post",
    // данные для отправки
    data,
    // элемент для вывода информации
    this.msgElement,
);
```
